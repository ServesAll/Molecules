## Set your app's theme by using the below markup

Add your own theme, if the key is the same as the default they will be overwritten

```code
import {ThemeWrapper} from @servesall/atoms;

const theme = {
    color1: '#FFFFFF', //colors
};

function RootElement() {
  return (
    <ThemeWrapper theme={theme}>
       <h3>This is JSX</h3>
    </ThemeWrapper>
  );
}
```

## Access theme

The access the theme props do as below

```code
import {useThemeContext} from @servesall/atoms;

function RamdomElement() {
    const {themeContext} = useThemeContext();
    const primaryColor = themeContext.color1;
}
```
